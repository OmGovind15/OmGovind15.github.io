CA-AFP: Cluster-Aware Adaptive Federated Pruning

This research proposes CA-AFP, a framework designed to optimize Federated Learning for resource-constrained edge devices by addressing both statistical and system heterogeneity. We combined the "Prune-and-Heal" strategy that allows models to self-heal by regrowing pruned weights during training along with the clustering which helps the model to personalise towards the client groups.

Novel contribution includes a hybrid importance scoring mechanism that analyzes weight magnitude, intra-cluster coherence, and gradient consistency. Experimental results on the UCI dataset demonstrated ~96 % accuracy with approximately 70% sparsity, similar in communication efficiency with pruning baselines while giving better performance and fairness(standard deviation across client's accuracies). We also conducted ablation studies which indicated the significance of each component of our framework and the usefullness of scoring formula while pruning in different FL scenarios.

Adaptive model outcome prediction and SHAP analysis in cricket ODI match

This project focused on predicting the outcomes of One Day International (ODI) cricket matches with high accuracy. We worked on making a robust set of features which can efficiently represent information of any particular over of a match.

Using these set of features we explored various models namely- Logistic Regression, SVM, Random Forest, XGBoost, LDA, Naive Bayes and Neural Network. Since the nature of the game is driven very specifically by innings, we experimented with various settings like training an individual model for all overs in a particular innings, training models individually for each over, and also having the switching model strategy over-wise implemented.

We observed that in the first innings, training a single model across all overs is much more effective as the winning chances are not sharply affected by any particular over event; it is mostly dependent on the past winning records of the team, which is also clearly visible in the SHAP plot for the 1st innings. However, in the second innings, the match dynamics change very quickly and can be captured better by training models for individual overs, essentially dividing a bigger problem into smaller sub-problems. We could also see the dynamic feature increasing effect in the 2nd innings. This work is useful for creating more specialized and precise predictive tools for sports analytics.

Crater and Boulder Detection on Planetary Surfaces

This project automates the identification of geological features from planetary images, a critical task for rover navigation and scientific analysis. We implemented a computer vision pipeline using the Canny edge detector and Circular Hough Transform to accurately locate craters and boulders. This automated approach is far more efficient than manual analysis, enabling faster mapping of extraterrestrial surfaces and aiding in the selection of safe landing sites for future missions.

Image Analysis with SIFT, BoW, and HOG

This project delves into classical feature-based image classification methods. We implemented the Scale-Invariant Feature Transform (SIFT) to extract robust keypoints from images, which were then clustered into a "Bag of Visual Words" (BoW). By creating histograms from these visual words, alongside Histogram of Oriented Gradients (HOG) features, we built a powerful model for classifying images. This work is fundamental to understanding the building blocks of many modern computer vision systems.

Optimal Design of Serosurveys during Pandemic

Under the supervision of Dr. Rajesh Sundaresan at IISc Bengaluru, this project focused on the optimal design of serosurveys for disease burden estimation.
